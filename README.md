# 🌍 Digital Clock - Multiple Time Zones

A beautiful, real-time digital clock that displays the current time across 12 different time zones around the world.

## Features

- ⏰ Real-time updates every second
- 🌎 12 different time zones:
  - **USA**: New York (EST), Chicago (CST), Denver (MST), Los Angeles (PST)
  - **Europe**: London (GMT), Paris (CET)
  - **Asia**: Mumbai (IST), Tokyo (JST), Singapore (SGT), Hong Kong (HKT)
  - **Pacific**: Sydney (AEST), Auckland (NZST)
- 🎨 Beautiful gradient background and card design
- 📱 Fully responsive and mobile-friendly
- ✨ Smooth animations and hover effects
- 📅 Displays date in addition to time
- ⏳ Shows AM/PM period

## How to Use

1. Open `index.html` in any web browser
2. Watch all 12 clocks update in real-time
3. No dependencies required - pure HTML, CSS, and JavaScript

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and animations
- `script.js` - Clock logic and timezone calculations
- `README.md` - This file
